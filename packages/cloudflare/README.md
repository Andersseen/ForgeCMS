# @forge-cms/cloudflare

Cloudflare-native bindings and adapter types for ForgeCMS.

## Target services

| Service | Binding type | Usage |
|---------|-------------|-------|
| **D1** | `D1Database` | SQL database (SQLite) |
| **R2** | `R2Bucket` | Object storage (S3-compatible) |
| **KV** | `KVNamespace` | Key-value cache & config |

## Status

**Structure only** — adapter implementations will be added in a future phase.
